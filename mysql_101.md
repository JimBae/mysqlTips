## MySQL

### root user 로 mysql 실행하기 <br>
 (-uroot : root user, -p : password) 
```
$ /usr/local/mysql/bin/mysql -uroot -p
```

### Create db
```
$ CREATE DATABASE dbtest;
```

### Check current databases
```
$ SHOW DATABASES;
```

### Use db
```
$ USE db_name;
Database changed
```

### Show tables
```
$ SHOW TABLES;
```

### 구조 tables
```
$ DESC tableName;
```
